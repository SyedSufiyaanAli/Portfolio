CW-Middleware

Project information

  • Category: Web Application
  • Client: College Project
  • Project date: 01 March, 2022

Lead role in the System analysis, Design, and development of an ETL layer to link two ERP systems CW1 and SAP. This solution has following features:

  • • Extract and strip Universal Shipment XML from CW1 and save information into SQL server database.
  • • Developed website - CRUD application to view, add or amend any missing shipment data before being sent to SAP system.
  • • Transform/Map data points between CW1 & SAP systems based on business logic.
  • • Generate final XML file as per target system spec and upload the same to SAP system.
  • • Windows schedule jobs created to:
    1. Continuously receive input XML files from CW1
    2. Strip input XML
    3. Create final output file to be sent to target system
Development Tools: C# .Net (MVC tech), HTML, JavaScript, SQL Server
